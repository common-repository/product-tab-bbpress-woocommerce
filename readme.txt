Product tabs Made Easy! bbPress Product Tab is a simple way to show forums into the WooCommerce Product Page. 

== Description ==

= Introduction =

bbPress Product Tab Plugin helps you to easily customize product tabs into your WooCommerce Store. It is super simple!

<blockquote>

= Features =
<ul>
 <li>Customize Product tab with bbPress forums.</li>
 <li>Customize the Reply link depends on User types.</li>
 <li>Add tabs Settings globally for all products, or on per product basis.</li>
 <li>Sort and rename Product tabs.</li>
 <li>Customize Product tab with view Product and reply options.</li> 
 <li>Set limit to show topics and replies.</li>
</ul>


</blockquote>

Highlights: Simply add forums on the product page, Customer Replies get via Product Page, Best product tab Plugin for bbPress in Market.

= How does it work? =

bbPress Product Tab helps you to easily customize product tabs into your WooCommerce Store. It is super simple!

Please refer the <a rel="nofollow" href="http://www.wpfloor.com/setting-product-tab-bbpress/">documentation</a> for more detailed help.

== Installation ==

1. Upload the plugin folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. That's it ! you can now configure the plugin.

== Frequently Asked Questions ==

= Does this plugin show Selected forums in product page? =

Yes. You can add the Specific forum for specific Product.
